SMTP By Company

This module will allow to configure outgoing mail server by company.

Outgoing mail server configuration by company

- Added field named Company
- You can create the outgoing mail server by specified by selected comapny.
- Here by selecting company, every email will be sent with reference to selected company, rather than default odoo mail system.
